## ПРАВКА 1 — Курсор там, где тапнул (Preview → Edit)

### Файл: `lib/features/notes/view/note_editor_screen.dart`

### Что сейчас

`_enterEditModeAtPosition` игнорирует `localPosition` и вызывает `_enterEditMode()`,
который кладёт курсор в конец текста:
```dart
void _enterEditModeAtPosition(BuildContext context, Offset localPosition) {
  _enterEditMode(); // ← localPosition ИГНОРИРУЕТСЯ
}

void _enterEditMode() {
  setState(() => _preview = false);
  WidgetsBinding.instance.addPostFrameCallback((_) {
    _contentCtrl.selection = TextSelection.collapsed(offset: text.length); // ← конец
    _contentFocus.requestFocus();
  });
}
```

### Решение

**Шаг 1.** Добавить переменные в класс `_NoteEditorScreenState`:
```dart
Offset? _pendingTapPosition;
final _previewKey = GlobalKey();
```

**Шаг 2.** Заменить `_enterEditModeAtPosition`:
```dart
void _enterEditModeAtPosition(BuildContext context, Offset localPosition) {
  _pendingTapPosition = localPosition;
  setState(() => _preview = false);
  WidgetsBinding.instance.addPostFrameCallback((_) {
    _placeCursorAtPendingPosition();
  });
}

void _placeCursorAtPendingPosition() {
  final tapPos = _pendingTapPosition;
  _pendingTapPosition = null;

  if (tapPos == null) {
    _contentFocus.requestFocus();
    return;
  }

  final text = _contentCtrl.text;
  if (text.isEmpty) {
    _contentFocus.requestFocus();
    return;
  }

  final previewBox = _previewKey.currentContext?.findRenderObject() as RenderBox?;
  final totalHeight = previewBox?.size.height ?? 600.0;
  final fraction = (tapPos.dy / totalHeight).clamp(0.0, 1.0);
  int finalOffset = (fraction * text.length).round().clamp(0, text.length);

  // Привязываем к границе слова
  while (finalOffset > 0 &&
      finalOffset < text.length &&
      !RegExp(r'[\s\n]').hasMatch(text[finalOffset - 1])) {
    finalOffset--;
  }

  _contentCtrl.selection = TextSelection.collapsed(offset: finalOffset);
  _contentFocus.requestFocus();
}
```

**Шаг 3.** Добавить `key: _previewKey` на GestureDetector в `_buildPreview`:
```dart
return GestureDetector(
  key: _previewKey, // ← добавить
  onTapDown: (details) =>
      _enterEditModeAtPosition(context, details.localPosition),
  // ...
);
```

**Шаг 4.** В `_enterEditMode()` убрать принудительный сдвиг курсора в конец:
```dart
void _enterEditMode() {
  _pendingTapPosition = null;
  setState(() => _preview = false);
  WidgetsBinding.instance.addPostFrameCallback((_) {
    _contentFocus.requestFocus(); // без изменения selection
  });
}
```

### ⚠️ Аккуратно
- Точность ≈ ±2-3 строки. Это физическое ограничение: Markdown-виджет не
  хранит маппинг экранных координат → символ.
- `addPostFrameCallback` обязателен — `TextField` ещё не создан в момент `setState`.
- Не вызывать `_placeCursorAtPendingPosition()` напрямую из `setState`.

## ПРАВКА 2 — Иконки тегов вместо папок

### Файл: `lib/features/notes/view/notes_screen.dart`

### Что сейчас
`ListTile.leading` всегда показывает папку. Теги выведены как Chip-ы внутри `title` —
перегруженно и визуально неаккуратно.

### Решение — заменить `itemBuilder` полностью

```dart
itemBuilder: (ctx, idx) {
  final note = notes[idx];
  final title = note.title.isEmpty ? 'Без названия' : note.title;
  final subtitle = _extractSubtitle(note.content);

  // Синхронно из кэша (Правка 9)
  final tags = provider.tagsForNote(note.id);

  // LEADING: тег > папка > ничего
  Widget? leading;
  if (tags.isNotEmpty) {
    final primaryTag = tags.first;
    leading = CircleAvatar(
      radius: 18,
      backgroundColor: Color(primaryTag.colorValue).withValues(alpha: 0.12),
      child: Icon(Icons.label, color: Color(primaryTag.colorValue), size: 18),
    );
  } else {
    Map<String, dynamic>? folder;
    if (note.folderId != null) {
      final found = provider.folders.where((f) => f['id'] == note.folderId);
      folder = found.isNotEmpty ? found.first : null;
    }
    if (folder != null) {
      leading = CircleAvatar(
        radius: 18,
        backgroundColor:
            Color(folder['color_value'] ?? 0xFFBDBDBD).withValues(alpha: 0.12),
        child: Icon(
          Icons.folder_outlined,
          color: Color(folder['color_value'] ?? 0xFFBDBDBD),
          size: 18,
        ),
      );
    }
  }

  // TITLE: если есть теги — "TagName · NoteTitle"
  String displayTitle;
  if (tags.isNotEmpty) {
    final tagName = tags.first.name;
    displayTitle = (note.title.isEmpty || note.title == tagName)
        ? tagName
        : '$tagName · ${note.title}';
  } else {
    displayTitle = title;
  }

  // SUBTITLE: контент + остальные теги
  final parts = <String>[];
  if (subtitle.isNotEmpty) parts.add(subtitle);
  if (tags.length > 1) {
    parts.add(tags.skip(1).map((t) => '#${t.name}').join(' '));
  }
  final subtitleText = parts.isNotEmpty ? parts.join('  ·  ') : null;

  return ListTile(
    key: ValueKey(note.id),
    leading: leading,
    title: Text(displayTitle,
        maxLines: 1,
        overflow: TextOverflow.ellipsis,
        style: const TextStyle(fontWeight: FontWeight.w600)),
    subtitle: subtitleText != null
        ? Text(subtitleText, maxLines: 1, overflow: TextOverflow.ellipsis)
        : null,
    trailing: Text(_formatDate(note.updatedAt),
        style: TextStyle(fontSize: 12, color: cs.secondary)),
    onTap: () => _openEditor(Navigator.of(context), provider, note),
    onLongPress: () => _showNoteOptions(note),
  );
},
```

### ⚠️ Аккуратно
- Убрать `FutureBuilder` — используем синхронный `provider.tagsForNote()` (Правка 9).
- Без Правки 9 список будет делать async-запрос на каждый элемент при каждом ребилде.

```dart
int _slashCommandStart = -1;

void _checkSlashCommand() {
  final text = _contentCtrl.text;
  final sel = _contentCtrl.selection;
  if (!sel.isValid || !sel.isCollapsed) {
    _closeCommandPalette();
    return;
  }
  final pos = sel.baseOffset;
  if (pos < 1) {
    _closeCommandPalette();
    return;
  }

  // Найти начало слова от позиции курсора назад
  int start = pos - 1;
  while (start > 0 &&
      text[start - 1] != '\n' &&
      text[start - 1] != ' ') {
    start--;
  }
  final word = text.substring(start, pos);

  if (word.startsWith('/') && !word.startsWith('//')) {
    final query = word.substring(1).toLowerCase();
    _slashCommandStart = start;
    _showCommandPalette(query);
  } else {
    _closeCommandPalette();
    _slashCommandStart = -1;
  }
}
```

```dart
void _showCommandPalette(String query) {
  final filtered = _kCommands.where((cmd) {
    return cmd.label.toLowerCase().contains(query) ||
        cmd.alias.any((a) => a.startsWith(query));
  }).toList();

  if (filtered.isEmpty) {
    _closeCommandPalette();
    return;
  }

  // Обновить если уже открыта (для live-фильтрации)
  if (_commandPaletteOverlay != null) {
    _commandPaletteOverlay!.markNeedsBuild();
    return;
  }

  _commandPaletteOverlay = OverlayEntry(
    builder: (ctx) {
      final cs = Theme.of(ctx).colorScheme;
      final bottomInset = MediaQuery.of(ctx).viewInsets.bottom;
      return Positioned(
        left: 8,
        right: 8,
        bottom: bottomInset + 4,
        child: Material(
          elevation: 8,
          borderRadius: BorderRadius.circular(12),
          color: cs.surfaceContainerHigh,
          child: ConstrainedBox(
            constraints: const BoxConstraints(maxHeight: 240),
            child: ListView(
              shrinkWrap: true,
              padding: const EdgeInsets.symmetric(vertical: 4),
              children: filtered
                  .map((cmd) => InkWell(
                        onTap: () {
                          _closeCommandPalette();
                          // Удалить "/query" из текста
                          if (_slashCommandStart >= 0) {
                            final t = _contentCtrl.text;
                            final pos = _contentCtrl.selection.baseOffset;
                            _contentCtrl.value = TextEditingValue(
                              text: t.substring(0, _slashCommandStart) +
                                  t.substring(pos.clamp(0, t.length)),
                              selection: TextSelection.collapsed(
                                  offset: _slashCommandStart),
                            );
                          }
                          _executeCommandById(cmd.id);
                        },
                        borderRadius: BorderRadius.circular(8),
                        child: Padding(
                          padding: const EdgeInsets.symmetric(
                              horizontal: 12, vertical: 10),
                          child: Row(
                            children: [
                              Icon(cmd.icon, size: 18, color: cs.primary),
                              const SizedBox(width: 10),
                              Expanded(
                                child: Text(cmd.label,
                                    style: const TextStyle(fontSize: 14)),
                              ),
                              Text(
                                '/${cmd.alias.first}',
                                style: TextStyle(
                                  fontSize: 11,
                                  fontFamily: 'monospace',
                                  color: cs.onSurface.withValues(alpha: 0.4),
                                ),
                              ),
                            ],
                          ),
                        ),
                      ))
                  .toList(),
            ),
          ),
        ),
      );
    },
  );
  Overlay.of(context).insert(_commandPaletteOverlay!);
}

void _closeCommandPalette() {
  _commandPaletteOverlay?.remove();
  _commandPaletteOverlay = null;
}

void _executeCommandById(String id) {
  switch (id) {
    case 'bold':     _toggleBold(); break;
    case 'italic':   _toggleItalic(); break;
    case 'strike':   _toggleStrikethrough(); break;
    case 'code':     _toggleCode(); break;
    case 'h1':       _toggleH1(); break;
    case 'h2':       _toggleH2(); break;
    case 'h3':       _toggleH3(); break;
    case 'ul':       _toggleBulletList(); break;
    case 'ol':       _toggleNumberedList(); break;
    case 'checkbox': _toggleCheckbox(); break;
    case 'quote':    _toggleBlockquote(); break;
    case 'hr':       _toggleHorizontalRule(); break;
    case 'bibleref': _insertBibleRef(); break;
    case 'notelink': _insertNoteLink(); break;
  }
}
```

static final _kMarkdownPatterns = [
  RegExp(r'^#', multiLine: true),
  RegExp(r'^\s*[-+*]\s', multiLine: true),
  RegExp(r'^\d+\.\s', multiLine: true),
  RegExp(r'\[.*\]\(.*\)'),
  RegExp(r'`[^`]+`'),
  RegExp(r'^> ', multiLine: true),
  RegExp(r'```'),
  RegExp(r'\*\*[^*]+\*\*'),
  RegExp(r'~~[^~]+~~'),
];

bool _looksLikeMarkdown(String text) =>
    _kMarkdownPatterns.any((p) => p.hasMatch(text));

// В _attachListeners добавить:
String _prevTextForPaste = '';
void _pasteDetector() {
  final current = _contentCtrl.text;
  // Большой прирост текста = вставка
  if (current.length - _prevTextForPaste.length > 50) {
    final inserted = current.substring(
        _prevTextForPaste.length.clamp(0, current.length));
    if (_looksLikeMarkdown(inserted)) {
      _showMarkdownPasteHint();
    }
  }
  _prevTextForPaste = current;
}

void _showMarkdownPasteHint() {
  if (!mounted) return;
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(
      content: const Text('Вставлен Markdown — он виден в предпросмотре'),
      action: SnackBarAction(
        label: 'Preview',
        onPressed: () {
          FocusScope.of(context).unfocus();
          setState(() => _preview = true);
        },
      ),
      duration: const Duration(seconds: 3),
    ),
  );
}

```dart
List<({int level, String text, int offset})> _extractOutline(String content) {
  final result = <({int level, String text, int offset})>[];
  final lines = content.split('\n');
  int offset = 0;
  for (final line in lines) {
    final m = RegExp(r'^(#{1,3})\s+(.+)').firstMatch(line);
    if (m != null) {
      result.add((
        level: m.group(1)!.length,
        text: m.group(2)!.trim(),
        offset: offset,
      ));
    }
    offset += line.length + 1;
  }
  return result;
}
```

```dart
Future<Map<String, List<NoteTag>>> getAllNoteTagsMap() async {
  final db = await _getDb();
  final rows = await db.rawQuery('''
    SELECT nt.note_id, t.id, t.name, t.color_value
    FROM note_tags nt
    JOIN tags t ON t.id = nt.tag_id
  ''');

  final result = <String, List<NoteTag>>{};
  for (final row in rows) {
    final noteId = row['note_id'] as String;
    final tag = NoteTag(
      id: row['id'] as String,
      name: row['name'] as String? ?? '',
      colorValue: row['color_value'] as int? ?? 0xFF2196F3,
    );
    result.putIfAbsent(noteId, () => []).add(tag);
  }
  return result;
}
```

```dart
List<NoteModel> getBacklinks(String noteId, String noteTitle) {
  if (noteTitle.isEmpty) return [];
  final titleLower = noteTitle.toLowerCase();
  return _notes.where((n) {
    if (n.id == noteId) return false;
    final contentLower = n.content.toLowerCase();
    return contentLower.contains('[[$titleLower]]') ||
        n.content.contains('[[$noteTitle]]');
  }).toList();
}
```
